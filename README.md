# Chat App

We are working on a chat application, focused on CI/CD with git flow branching strategy and left shift testing.

## Tech-stack

FullStack App that uses .NET Web API, REACT, and PostgreSQL.

- Programming Languages:
  - Backend: C# AspNetCore
  - Frontend: TypeScript
  - Database: SQL

- Database Engines:
  - Neon.com
  - Redis

## Architecture

Frontend is built with REACT (WIP) and backend is built with ASP.NET Web API (WIP)
Data is stored in Neon.com in PostgreSQL (WIP)
